Javascript几种创建对象的方法

1.工厂方法

优点：创建相同实例只有一处代码。
缺点：不知道对象原型。

2.构造函数

优点：可以知道实例原型。
缺点：方法不是同一个方法实例。

3.原型方法

优点：共享方法实例对象。
缺点：每个实例需要定义非方法属性。

4.构造函数和原型的组合方法

优点：非方法属性在构造函数中定义，方法属性则在原型中定义。

总结：最后一种方法是比较好的创建对象的方式，综合了前面3种方式的优点。每个实例对象都有自己实例属性的一份副本，但同时共享着方法的引用，最大限度节省了内存。
